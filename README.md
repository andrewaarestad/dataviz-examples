# dataviz-examples

This repo contains a Next.js app with various examples of data visualizations using different libraries and frameworks.

## Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
